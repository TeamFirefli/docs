## Summary

Permanently delete a recommendation. This action cascades to remove all votes and comments associated with the recommendation.

<Warning>
  **Destructive Action:** This permanently deletes the recommendation and cannot be undone. All votes and comments will also be deleted.
</Warning>

## Important Notes

- Requires `deletedById` field to track who performed the deletion
- All associated votes are automatically deleted
- All associated comments are automatically deleted
- Consider using status `archived` instead if you need to preserve history
