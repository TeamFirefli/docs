## Summary

Fetch current activity for a specific user, including session duration, message count, and detailed session logs from the last reset date till current.

### Request

```
GET https://app.firefli.net/v1/workspace/{workspace_id}/user/{user_id}/pactivity
```

#### Query Parameters

| **Parameter** | **Type** | **Description**                                   |
| :------------ | :------- | :------------------------------------------------ |
| limit         | number   | Items per page (default: 50, max: 100)            |
