## Summary

Return leaderboard rankings for the specified workspace, including top players by playtime and session count. Use this endpoint to display competitive rankings and track your position.

#### Path Parameters

| **Parameter** | **Type** | **Description**      |
| :------------ | :------- | :------------------- |
| workspaceId   | string   | The workspace ID     |

#### Query Parameters

| **Parameter** | **Type** | **Description**                                   |
| :------------ | :------- | :------------------------------------------------ |
| userId        | number   | Specific users stats and position                 |


### Response example

```json
{
  "playtime": {
    "top_three": [
      {
        "_id": "693938",
        "position": 1,
        "total": 60,
        "in_game": false
      }
    ]
  },
  "you": {
    "_id": "693938",
    "position": 1,
    "total": 60,
    "in_game": false
  },
  "sessions": {
    "top_three": [
      {
        "_id": "693938",
        "position": 1,
        "total": 0
      }
    ]
  }
}
```

<Info>
  The `playtime.total` field represents total playtime in minutes. The `you` object is only included when the `userId` query parameter is provided, showing that user's position on the leaderboard.
</Info>

### Code samples

<CodeGroup>
```http
GET https://app.firefli.net/v1/workspace/752856518/leaderboard?userId=693938
Authorization: Bearer firefli_your_api_key
```
```javascript
const leaderboard = await fetch(
  "https://app.firefli.net/v1/workspace/752856518/leaderboard?userId=693938",
  { headers: { Authorization: "Bearer firefli_your_api_key" } }
).then((r) => r.json());
console.log(leaderboard.you.position); // User's position
console.log(leaderboard.playtime.top_three); // Top 3 players
```
```python
import requests

leaderboard = requests.get(
    "https://app.firefli.net/v1/workspace/752856518/leaderboard",
    headers={"Authorization": "Bearer firefli_your_api_key"},
    params={"userId": "693938"}
).json()
print(leaderboard["you"]["position"])
print(leaderboard["playtime"]["top_three"])
```
