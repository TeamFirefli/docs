## Providers

### RankGun

RankGun drives advanced rank automation. Link your workspace with these steps:

<Steps>
  <Step>
    <h4>Enter API key</h4>
    <p>Enter the RankGun API key provided in your RankGun dashboard.</p>
  </Step>
  <Step>
    <h4>Provide Workspace ID</h4>
    <p>Provide the RankGun Workspace ID (e.g., <code>VDzWOQ1BqHnOeaa</code>).</p>
  </Step>
  <Step>
    <h4>Save and verify</h4>
    <p>Save the configuration and verify the status indicator turns green.</p>
  </Step>
</Steps>

<Warning>
  RankGun enforces request limits - consult their [API Documentation](https://www.rankgun.works/docs/developing/APIRefrence).
</Warning>

### Roblox Open Cloud

Roblox has changed how it's public api endpoints work.
This means it is now **required** for users to setup a Open Cloud API key in order to SYNC Members.

Additionally, this now means that you can natively rank with Firefli, without any additional services.

<Warning>
  Roblox recommends you create the API Key on an ALT Account, with a high ranking role. - Do NOT use a Group API Key.
</Warning>

<Steps>
  <Step>
    <h4>Create an API Key</h4>
    <p>Head to [create.roblox.com/dashboard/credentials](https://create.roblox.com/dashboard/credentials) and create an API Key, on an account/group with correct permissions.</p>
  </Step>
  <Step>
    <h4>Access Permissions</h4>
    <p>As a minimum, you'll need to add the Groups API with 'group:read' operations and Users API with 'user.social:read'.</p>
    <p>If you'd like to use ranking too, please add the 'write' operation.</p>
  </Step>
  <Step>
    <h4>Save & Generate Key</h4>
    <p>Select 'Copy Key to clipboard'.</p>
  </Step>
  <Step>
    <h4>Open Services Settings on Firefli</h4>
    <p>Paste in your API Key.</p>
    <p>Select 'Test Key' and ensure a 'Valid' response is returned.</p>
    <p>Press 'Save Settings' to ensure this is stored on your workspace.</p>
  </Step>
</Steps>
