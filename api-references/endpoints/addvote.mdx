## Summary

Add a vote to a recommendation. Users can only vote once per recommendation - duplicate votes return a 409 Conflict error.

## Important Notes

<Info>
  **Duplicate Prevention:** The API enforces a unique constraint ensuring each user can only vote once per recommendation.
</Info>

## Response

Returns the created vote and updated total vote count for the recommendation.
