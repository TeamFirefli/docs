Planetary Cloud will automatically refresh ranks every hour, to be sure that everyone that should have access does.
If you're selfhosting, see our [Automations](https://docs.planetaryap.us/deployment/automation) page for setup.

### Creating Roles
<Steps>
  <Step>
    <h4>Go to workspace settings</h4>
    <p>Access settings via the sidebar.</p>
  </Step>
  <Step>
    <h4>Click Permissions tab</h4>
    <p>Navigate to the Permissions section.</p>
  </Step>
  <Step>
    <h4>Expand Roles section</h4>
    <p>Find and expand the Roles configuration area.</p>
  </Step>
</Steps>

### Add New Role
<Steps>
  <Step>
    <h4>Sync external ranks</h4>
    <p>Click **"Check group roles"** to sync external ranks.</p>
  </Step>
  <Step>
    <h4>Add Role</h4>
    <p>Click **"Add Role"** to create a new role.</p>
  </Step>
  <Step>
    <h4>Configure the role</h4>
    <p>Expand the new role and configure:</p>
    <ul>
      <li>Role name</li>
      <li>Permissions</li>
      <li>Optional: Link to group rank for auto-assignment</li>
    </ul>
    <p>Changes save automatically.</p>
  </Step>
</Steps>

### Assign Roles
<Steps>
  <Step>
    <h4>Navigate to Users</h4>
    <p>Go to **Settings > Permissions > Users**.</p>
  </Step>
  <Step>
    <h4>Search for user</h4>
    <p>Search by Roblox username.</p>
  </Step>
  <Step>
    <h4>Assign role</h4>
    <p>Select role from dropdown. Changes apply instantly.</p>
  </Step>
</Steps>

### Group Sync
<Steps>
  <Step>
    <h4>Find Group-Synced Roles</h4>
    <p>In **Roles** tab, locate the **Group-Synced Roles** section.</p>
  </Step>
  <Step>
    <h4>Match ranks</h4>
    <p>Match Roblox group ranks to Orbit roles.</p>
  </Step>
  <Step>
    <h4>Check group ranks</h4>
    <p>Click **"Check group ranks"** if needed to sync.</p>
  </Step>
</Steps>

### Remove Users
<Steps>
  <Step>
    <h4>Go to Users section</h4>
    <p>Navigate to **Settings > Permissions > Users**.</p>
  </Step>
  <Step>
    <h4>Remove user</h4>
    <p>Click **Remove** button next to user. Access is revoked immediately.</p>
  </Step>
</Steps>
