## Summary

Fetch activity history for a specific user, including session duration, message count, and detailed session logs with date filtering.

### Request

```
GET https://api.planetaryapp.us/v1/workspace/{workspace_id}/user/{user_id}/activity
```

#### Query Parameters

| **Parameter** | **Type** | **Description**                                   |
| :------------ | :------- | :------------------------------------------------ |
| startDate     | ISO 8601 | Filter by start date (e.g., 2024-01-01T00:00:00Z) |
| endDate       | ISO 8601 | Filter by end date (e.g., 2024-01-31T23:59:59Z)   |
| page          | number   | Page number (default: 1)                          |
| limit         | number   | Items per page (default: 50, max: 100)            |
