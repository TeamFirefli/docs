### Setting up Firefli in your server

<Warning>
  You require basic knowledge in usage of terminal as well as NextJS.\
  Team Firefli is **not responsible** for any user errors caused during the setup process or usage.
</Warning>

### Prerequisites

- A machine that is capable of running Command/Console tasks (e.g. Ubuntu)
- Root access
- PostgreSQL Database
- Package manager (e.g. npm)
- Domain and SSL Encryption

### Installing Firefli

Once you're connected to your server, follow the command guide below in your terminal/console session.

<Steps>
  <Step>
    <h4>Update your Server</h4>

```
sudo apt update
sudo apt upgrade
```
  </Step>

  <Step>
    <h4>Install Required packages</h4>

```
sudo apt install git curl build-essential -y
```
  </Step>

  <Step>
    <h4>Install Node.js (v20+) and your package manager</h4>

```
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```
  </Step>

  <Step>
    <h4>Install PostgresSQL</h4>

```
sudo apt install postgresql postgresql-contrib -y
sudo systemctl enable --now postgresql
```
  </Step>

  <Step>
    <h4>Create PostgreSQL User and Database</h4>

```
sudo -u postgres psql
```

```
CREATE USER firefliuser WITH PASSWORD 'strongpassword';
CREATE DATABASE fflidb OWNER firefliuser;
\q
```

<Note>
  Ensure you replace "strongpassword" with your own unique password.
</Note>
  </Step>

  <Step>
    <h4>Clone Firefli Repository</h4>

```
git clone https://github.com/TeamFirefli/firefli && cd firefli
```
  </Step>

  <Step>
    <h4>Install Firefli Dependencies</h4>

<CodeGroup>

```npm npm
npm install
```


```pnpm pnpm
pnpm install
```


```yarn yarn
yarn install
```

</CodeGroup>
  </Step>

  <Step>
    <h4>Create `.env` configuration file</h4>

```
nano .env
```

<p>Paste the following:</p>

```
DATABASE_URL="postgresql://firefliuser:strongpassword@localhost:5432/fflidb" - Your connection string (e.g. hosted on Supabase, Railway, Neon, etc.)
SESSION_SECRET="" - A strong secret string (e.g. generated via openssl rand -base64 32)
ENCRYPTION_KEY="" - Encrypt your Discord intergration (e.g. generated via openssl rand -base64 64)
NEXTAUTH_URL="" - Your Domain/Connection URL (e.g. https://domain.domain.com - no / at end.)
```

<Tip>
  Remember to replace "strongpassword" with your chosen secure password from earlier.
  Generate a 32/64-character session key here: [1Password Generator](https://1password.com/password-generator/)
</Tip>

<p>**Save and close (Ctrl + X -\> Y -\> Enter)**</p>
  </Step>

  <Step>
    <h4>Initialize Database</h4>

```
npx prisma db push
```
  </Step>

  <Step>
    <h4>Build Firefli</h4>

<CodeGroup>

```npm npm
npm run build
```


```pnpm pnpm
pnpm run build
```


```yarn yarn
yarn run build
```

</CodeGroup>
  </Step>

  <Step>
    <h4>Start Firefli</h4>

<CodeGroup>

```npm npm
npm run start
```


```pnpm pnpm
pnpm run start
```


```yarn yarn
yarn run start
```

</CodeGroup>
  </Step>

  <Step>
    <h4>Run Firefli in Background with PM2 **(RECOMMENDED)**</h4>

```
npm install -g pm2
```

```
pm2 start npm --name="Firefli" -- start
```

```
pm2 startup
pm2 save
```

<Warning>
  Remember to build your database otherwise you won't be able to access your workspace.
</Warning>
  </Step>

  <Step>
    <h4>Complete setup via Web Interface</h4>

    <p>1. Go to https://your.domain.com - Must be SSL</p>
    <p>2. Enter your **Roblox Group ID**</p>
    <p>3. Enter your **Roblox username** and a password</p>
    <p>4. ðŸŽ‰ You're all set!</p>
  </Step>
</Steps>
