## Providers

### RankGun

RankGun drives advanced rank automation. Link your workspace with these steps:

<Steps>
  <Step>
    <h4>Enter API key</h4>
    <p>Enter the RankGun API key provided in your RankGun dashboard.</p>
  </Step>
  <Step>
    <h4>Provide Workspace ID</h4>
    <p>Provide the RankGun Workspace ID (e.g., <code>VDzWOQ1BqHnOeaa</code>).</p>
  </Step>
  <Step>
    <h4>Save and verify</h4>
    <p>Save the configuration and verify the status indicator turns green.</p>
  </Step>
</Steps>

<Warning>
  RankGun enforces request limits—consult their [API Documentation](https://www.rankgun.works/docs/developing/APIRefrence).
</Warning>
