## Summary

Get aggregated statistics showing the total count of notices broken down by status (pending, approved, rejected, active).

### Request

```
GET https://app.firefli.net/api/public/v1/workspace/{workspaceId}/notices/summary
```

#### Path Parameters

| **Parameter** | **Type** | **Description**      |
| :------------ | :------- | :------------------- |
| workspaceId   | string   | The workspace ID     |

### Response example

```json
{
  "success": true,
  "summary": {
    "total": 150,
    "pending": 5,
    "approved": 120,
    "rejected": 25,
    "active": 8
  }
}
```
