## Summary

Fetch all notices for a workspace with filtering options for status, user, dates, and pagination support.

### Request

```
GET https://app.firefli.net/api/public/v1/workspace/{workspaceId}/notices
```

#### Path Parameters

| **Parameter** | **Type** | **Description**    |
| :------------ | :------- | :----------------- |
| workspaceId   | string   | The workspace ID   |

#### Query Parameters

| **Parameter** | **Type** | **Description**                                              |
| :------------ | :------- | :----------------------------------------------------------- |
| status        | string   | Filter by status: `pending`, `approved`, `rejected`, `active` |
| userId        | number   | Filter by specific user ID                                    |
| from          | ISO 8601 | Filter by start date (e.g., 2024-01-01T00:00:00.000Z)        |
| to            | ISO 8601 | Filter by start date (e.g., 2024-01-31T23:59:59.999Z)        |
| limit         | number   | Number of results (default: 50, max: 100)                    |
| offset        | number   | Pagination offset (default: 0)                               |
