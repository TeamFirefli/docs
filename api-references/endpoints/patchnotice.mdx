## Summary

Modify a notice's status, add review comments, update end time, or revoke the notice.

#### Path Parameters

| **Parameter** | **Type** | **Description**      |
| :------------ | :------- | :------------------- |
| workspaceId   | string   | The workspace ID     |
| noticeId      | string   | The notice UUID      |

#### Request Body (all fields optional)

| **Field**      | **Type**  | **Description**                                   |
| :------------- | :-------- | :------------------------------------------------ |
| approved       | boolean   | Approve (true) or reject (false) the notice       |
| reviewComment  | string    | Comment from reviewer                             |
| endTime        | ISO 8601  | Update end date (null for indefinite)             |
| revoked        | boolean   | Revoke the notice                                 |

### Request example (Approve)

```json
{
  "approved": true,
  "reviewComment": "Approved"
}
```

### Request example (Reject)

```json
{
  "approved": false,
  "reviewComment": "Insufficient coverage during this period"
}
```
