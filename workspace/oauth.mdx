### Open Cloud
<Steps>
  <Step>
    <h4>Create OAuth Application</h4>
    <p>You'll require an active Open Cloud OAuth Application to link with Firefli. Create an application [here](https://create.roblox.com/dashboard/credentials?activeTab=OAuthTab).</p>
  </Step>
  <Step>
    <h4>Name your application</h4>
    <p>Name the application name something unique. This will be displayed to everyone that uses it so pick wisely!</p>
  </Step>
  <Step>
    <h4>Save credentials</h4>
    <p>Save both Client ID and Secret (You'll need this later).</p>
  </Step>
  <Step>
    <h4>Fill out details</h4>
    <p>Fill out the relevant details, you can use our Policy links. Replace 'instance' with your unique Instance name.</p>

```
Entry Link:
https://your.domain.com/login

Privacy Policy:
https://www.firefli.net/privacy

Terms of Service:
https://www.firefli.net/terms
```
  </Step>
  <Step>
    <h4>Select scopes</h4>
    <p>In the scopes section, select **OpenID** and **Profile**.</p>
  </Step>
  <Step>
    <h4>Setup redirect URL</h4>
    <p>Setup a redirect URL (e.g. https://your.domain.com/api/auth/roblox/callback).</p>
  </Step>
  <Step>
    <h4>Review and Publish</h4>
    <p>Press Review and Publish! This will trigger Roblox moderation to test your signin page.</p>
  </Step>
</Steps>
