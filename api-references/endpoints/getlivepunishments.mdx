## Summary

Poll for new punishments that should be enforced in-game. Returns resolved `kick`, `temp_ban`, and `perm_ban` cases created after the `since` timestamp that have an active `PlayerBan` record.

Expired and revoked bans are automatically excluded.

<Info>
  Call this endpoint on a schedule (e.g. every 60 seconds) and store the latest `resolvedAt` timestamp as your next `since` value.
</Info>

## Roblox Example

```lua
local HttpService = game:GetService("HttpService")

local BASE_URL = "https://app.firefli.net/api/public/v1/workspace/42/moderation"
local API_KEY  = "your_api_key"
local lastPoll = DateTime.now():ToIsoDate()

game:GetService("RunService").Heartbeat:Connect(function()
    -- poll every 60 seconds
end)

local function pollPunishments()
    local url = BASE_URL .. "/live?since=" .. lastPoll
    local ok, result = pcall(function()
        return HttpService:GetAsync(url, false, { Authorization = "Bearer " .. API_KEY })
    end)
    if not ok then return end
    local data = HttpService:JSONDecode(result)
    lastPoll = DateTime.now():ToIsoDate()
    for _, punishment in ipairs(data.data) do
        -- enforce punishment.action on punishment.userId
    end
end
```
