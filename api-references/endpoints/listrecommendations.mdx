## Summary

Retrieve all recommendations for a workspace with flexible filtering options including status, target user, creator, and pagination support. Perfect for building recommendation dashboards and management interfaces.

## Use Cases

- Display all active recommendations awaiting approval
- Filter recommendations by specific users
- Build analytics on recommendation patterns
- Export recommendation data for reporting
