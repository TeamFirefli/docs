## Summary

Submit a moderation case against a Roblox user. The case is logged to the audit trail and the target user is automatically upserted into the system if they don't already exist.

<Info>
  The `action` field records the intended punishment type. Use `perm_ban` for permanant bans, `kick` for kicks, `temp_ban` for temporary bans, or `warning` for warnings without enforcement.
</Info>

## Actions

| Value | Description |
|---|---|
| `warning` | Informal warning — no enforcement |
| `kick` | Kick from the game |
| `temp_ban` | Temporary Ban (use with or `expiresAt`) |
| `perm_ban` | Permanant Ban (use with `isPermanent`) |
