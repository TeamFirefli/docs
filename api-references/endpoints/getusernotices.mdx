## Summary

Fetch all notices associated with a specific user, with optional filtering by status and date range.

### Request

```
GET https://app.firefli.net/api/public/v1/workspace/{workspaceId}/notices/user/{userId}
```

#### Path Parameters

| **Parameter** | **Type** | **Description**      |
| :------------ | :------- | :------------------- |
| workspaceId   | string   | The workspace ID     |
| userId        | number   | The user ID          |

#### Query Parameters

| **Parameter** | **Type** | **Description**                                              |
| :------------ | :------- | :----------------------------------------------------------- |
| status        | string   | Filter by status: `pending`, `approved`, `rejected`, `active` |
| from          | ISO 8601 | Filter by start date (e.g., 2024-01-01T00:00:00.000Z)        |
| to            | ISO 8601 | Filter by start date (e.g., 2024-01-31T23:59:59.999Z)        |
